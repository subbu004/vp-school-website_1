## Project Structure

```
vidhya-parthi/
│
├── index.html                  ← Main HTML (Phase 1: Header + Hero + Footer)
│
├── css/
│   └── style.css               ← Base styles, theme variables, responsive design
│
├── js/
│   └── script.js               ← Base JS: navbar, hamburger, smooth scroll, scroll-to-top
│
├── images/
│   ├── logo.png                ← School logo (replace placeholder)
│   ├── hero-bg.jpg             ← Hero background image (add your image)
│   ├── favicon.png             ← 32×32 favicon
│   ├── apple-touch-icon.png    ← 180×180 iOS icon
│   └── og-image.jpg            ← 1200×630 Open Graph image
│
└── README.md
```

## Getting Started

1. Replace logo.png with the actual school logo
2. Add images/hero-bg.jpg (recommended: 1920x1080, school campus photo)
3. Open index.html in a browser — no build step required
4. Update contact details in the footer
